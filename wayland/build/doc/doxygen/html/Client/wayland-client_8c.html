)
static

This function is called for error events and indicates that in some object an error occurred.

The difference between this function and display_fatal_error() is that this one handles errors that will come by wire, whereas display_fatal_error() is called for local errors.

Parameters
display
codeerror code
idid of the object that generated the error
intfprotocol interface
