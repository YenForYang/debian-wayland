void(* wl_shell_interface::get_shell_surface) (struct wl_client *client, struct wl_resource *resource, uint32_t id, struct wl_resource *surface)

create a shell surface from a surface

Create a shell surface for an existing surface. This gives the wl_surface the role of a shell surface. If the wl_surface already has another role, it raises a protocol error.

Only one shell surface can be associated with a given surface.

Parameters
idshell surface to create
surfacesurface to be given the shell surface role
